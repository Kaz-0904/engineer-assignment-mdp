# 技術者派遣業界における技術者1人/複数人を管理対象とした案件アサインの逐次意思決定問題に関する研究
このリポジトリは、技術者派遣企業における人材割り当ての逐次意思決定問題についてMDPで定式化し、シミュレーション可能にしたものです。

## 概要
本リポジトリは個人の学習・研究目的で作成しています。
本プロジェクトの目的は、強化学習の研究への前準備として、既存研究の調査を飛ばして、論文執筆のプロセスを一通り経験することです。
1人技術者の割り当て、複数技術者の割り当ての定式化へ取り組み、練習用論文を書き終わるところまでをゴールとします。

## 備考(自分用のメモ)
「現実問題への応用論文」に取り組み中。
次は強化学習が適用されている産業・ビジネス、研究について調べる。

一旦中断。「とりあえずMDPの形にして、論文の問いや中身は後で考える」という進め方にしたのが失敗の原因だった。事前に検証したい問いを決めておく必要がある。理由としては、シミュレーションはいくらでもパラメータを変えて結果を出すことができるので、後に考えようとして膨大な結果の前に立ち尽くすことになってしまった。次にシミュレーション系の研究を考える場合には、必ず、シミュレーションを動かす前に検証したい内容をYes/Noの分かる形で言葉に落としておく必要がある。
