# AGENTS.md

このリポジトリは、技術者派遣における単一技術者の案件アサイン問題を
有限状態MDPとして実装する研究用コードである。

## 目的
- 数式ノートの内容をコードに落とす
- 価値反復法・方策反復法・ベースライン方策を比較する
- Colab上で再現可能にする
- 将来的な感度分析・拡張に耐える構成にする

## 重視すること
- 数式との対応が明確
- 可読性
- パラメータ変更のしやすさ
- 遷移確率と報酬の定義が一箇所にまとまっていること
- 最小で正しく動くこと

## 避けること
- 状態や行動の定義がファイル内に散在すること
- 研究ノートにない仕様を無断で追加すること
- 過剰な抽象化
- 使い方が分かりにくい構成

## 実装方針
- Python + numpy + matplotlib を基本とする
- コメントは日本語で丁寧に書く
- notebook から一発で主要結果が再現できるようにする
